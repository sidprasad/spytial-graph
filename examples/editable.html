spytial-graph — editable embedding

Render a graph you can examine and update in a plain static page. Use the toolbar (+ Node, + Relation) or drag between nodes to edit; constraints re-solve live. The panels on the right update on every edit — the notation is re-derived from the edited graph (handle.getSource()), and the value is its reified JSON (handle.getValue()). That round-trip is the whole point.
